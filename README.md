# SnehaLanjewar.github.io

Personal portfolio website hosted with **GitHub Pages**.

Live site: https://snehalanjewar.github.io/

---

## About

This repository contains a simple static portfolio site built with **HTML + CSS** and published using **GitHub Pages**.  
The homepage includes sections like:

- About Me
- Professional Journey
- Project Highlights
- Skills
- Contact

---

## Tech Stack

- **HTML5**
- **CSS3** (inline styles in `index.html`)
- **GitHub Pages**
- **Jekyll theme config**: `jekyll-theme-slate` (via `_config.yml`)

---

## Project Structure

- `index.html` — Main static portfolio page
- `_config.yml` — GitHub Pages / Jekyll configuration
- `upload.gif` — Asset used in the repo (optional)
